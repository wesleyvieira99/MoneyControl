🔑 Configuração da IA (OpenAI)

A chave da API OpenAI deve ser configurada no servidor via variável de ambiente OPENAI_API_KEY. Não armazenamos sua chave no frontend por segurança.

⚠️ Segurança:

Nunca exponha sua chave API no código-fonte ou frontend. Configure via:
export OPENAI_API_KEY=sk-proj-...
e reinicie o backend.
